Developers Rules

  • Make sure all code is in the Hackathon repository. Create a sub folder under Dec2017 with your team code. You can upload it close to finishing to avoid any merge conflicts / accidentally deleting someone else's code.
  • Do not make any modifications on staging or production, or teams will face disqualification.
  • Do not use existing staging / production keys if you are using 3rd party tools like Twilio / Nexmo / Google etc.
